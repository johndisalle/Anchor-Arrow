# Color Assets — Anchor & Arrow

Add these named colors to `Assets.xcassets` in Xcode.
Create a Color Set for each name below with light/dark values.

---

## Brand Colors

### BrandAnchor (Deep Blue-Slate — rootedness)
- Light: #2C5F8A  (strong, trustworthy blue)
- Dark:  #4A90C4  (slightly lighter for dark mode)

### BrandArrow (Warm Amber/Gold — purposeful movement)
- Light: #D4762A  (earthy amber)
- Dark:  #F0973D  (brighter amber for dark mode)

### BrandEarth (Rich Earth Brown — tree trunk/roots)
- Light: #6B4A2A
- Dark:  #8A6A4A

### BrandGold (Achievement Gold)
- Light: #C9A227
- Dark:  #F0C040

### BrandWarning (Drift/Alert Orange-Red)
- Light: #C0392B
- Dark:  #E74C3C

### BrandDanger (Error Red)
- Light: #B91C1C
- Dark:  #EF4444

## How to add in Xcode:
1. Open Assets.xcassets
2. Right-click → New Color Set
3. Name it exactly as above (e.g., "BrandAnchor")
4. Click "Any Appearance" and "Dark" squares to set light/dark values
5. Enter hex values using the Color picker → Hex Color option
